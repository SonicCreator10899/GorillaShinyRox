GORILLA SHINY ROX - STABLE NO-IMAGES PUBLIC BETA

Decision:
- Cosmetic images were removed for stability.
- Fandom image URLs / WPF image decoding were unreliable.
- A public beta should be reliable before it is fancy.

What remains:
- Load All Current Cosmetics
- Text-only cosmetic catalog
- Shiny Rox prices
- Free cosmetics
- Creator badges / not buyable
- Price unknown fallback
- Use selected price as goal
- Current Shiny Rox tracker
- Start-at-zero timer
- Estimated unlock date
- Tips panel

Why no images:
- Images caused broken thumbnails and compile/dependency issues.
- They can come back later in v0.2 after the core app is stable.

Build:
1. Open GorillaShinyRox.csproj in Visual Studio.
2. Build > Clean Solution.
3. Build > Rebuild Solution.
4. Run.
5. Click Load All Current Cosmetics.
